# ToDo Web App

A simple ToDo web application built with vanilla JavaScript and HTML.

## Project overview

This project includes a basic task manager interface with the following features:

- Add new tasks
- View tasks in a list
- Clean, simple UI layout

## Files

- `index.html` — main webpage structure
- `app.js` — application logic

## Run locally

1. Open `index.html` in your browser.

## Notes

This is a lightweight demo project for learning front-end development with JavaScript and HTML. You can extend it by adding task persistence, editing, and delete functionality.
